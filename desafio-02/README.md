# AWS Step Functions

- O serviço AWS Step Functions, é um serviço de orquestração de fluxos de trabalho, que permite integra e automatizar serviços da AWS, como o (lambda, S3, SNS, SQS e DynamoDB), de uma maneira visual e pouco código um serviçõ LOW CODE da AWS.

- Através dele podemos executar workflows para validar arquivos, acionar funções, coordenar serviços e monitorar execuções, aplicando conceitos de automação, mensageria e boas práticas de arquiterura em nuvem.

- Ele permite transformar uma lógica de negócios complexa em fluxos de trabalhos visuais e claros, por meio deu ma interface interativa de drag-and-drop. Isso acelera o desenvolvimento e simplifica a solução de problemas.

- Nós podemos usar uma arquitetura sem servidor para lidar com workloads cada vez maiores e com eficiência.
